# Система управления личной библиотекой

Консольное приложение для управления книгами и пользователями библиотеки, разработанное на C++ с использованием ООП.

##  Функционал

- Управление каталогом книг (добавление, просмотр, поиск)
-  Регистрация и управление пользователями
-  Выдача и возврат книг
-  Автоматическое сохранение и загрузка данных
-  Интуитивное консольное меню

##  Требования

- C++17
- CMake 3.10+
- Компилятор с поддержкой C++17 (g++, clang, MSVC)

## Сборка и запуск

### Linux/Mac:
```bash
git clone <ваш-репозиторий>
cd library_project
mkdir build && cd build
cmake ..
make
./library
